# PRD Writing Skill

This skill provides templates, guidelines, and examples for writing effective Product Requirements Documents (PRDs).

## Purpose

- **Template structure** - Standard 8-section PRD format
- **Writing guidelines** - How to write each section effectively
- **Demo-grade focus** - Optimized for MVPs and demos, not enterprise ceremony

## When to Use This Skill

Use this skill when:
- Generating a new PRD from an idea
- Clarifying or refining an existing PRD
- Reviewing a PRD for completeness
